• clash Subscription Link to sing-box

Tool for converting Clash format subscription links to sing-box. The converted link can be used for the Android version of SFA, iOS version not tested.

Paste the subscription link generated by this tool on this page (only parameters saved in the link are available), which can be resolved to the options below


Configuration File Template
You can customize the nodes included in the generated configuration file by setting outbounds for urltest/selector. For example: 
{
    "tag": "selector",
    "type": "selector",
    "outbounds": [
        "include: Singapore",
        "exclude: Hong Kong"
    ]
}
